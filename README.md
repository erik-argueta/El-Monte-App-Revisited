# El Monte Notifier (Re-Visited)

## Description
A mobile app that will notify El Monte Residents of upcoming official events.  
This project is ever evolving to bridge the communities accessibility to the resources they seek.  

### Modules
* Official Announcements
* Food Drive Events
* Town Hall Events
* Calendar

## Accessibility Options
This app will have the following languages that are common among the community:
* English
* Spanish
* Mandarin
* Vietnamese


## Purpose
El Monte provides notifications for their public events through e-mails and their social media profile(s). This app aims to merge their different avenues into a single source.  

## Stack
I initially decided to build the app in Flutter due to its robustness with creating Web Applications and Mobile Applications.

### Front-End
* Flutter, dart
    * Swift
    * Kotlin

### Back-End
* Firebase
